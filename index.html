Heading Tag

There are six different heading tags that can be used. These range from <h1>, <h2>,...., <h6>. Each tag would be used under different circumstances as each tag represents a different heading size. For instance, <h1> would be used for the most important heading (main heading) as it appears the largest whereas <h2> would be used for the next important heading (sub heading).

The following code snippet represents how heading tags can be utilized:

<h1> Main Heading Here </h1>
.
.
<h2> Sub Heading Here </h2>
<h2> Sub Heading Here </h2>

Paragraph Tag

The paragraph tag or <p> is used for paragraph contents. Paragraph contents are represented in a form of text.

The following code snippet represents how paragraph tags can be utilized:

<p> Text! </p>

Anchor Tag

The anchor tag or <a> is used to connect to an external webpage by defining a hyperlink. Important attributes of the anchor tag include href and target. The href attribute is used to define the webpage destination. The target attribute is used to determine if the webpage destination opens in a new tab or replaces the current webpage in the same tab.

The following code snippet represents how anchor tags can be utilized:

<a href="www.google.com"> Google.com in same tab </a>
<a href="www.google.com" target="_blank"> Google.com in a new tab </a>

List Tag

There are three different list tags that can be used. An ordered list or <ol> is a list that uses numbered bullets. An unordered list or <ul> is a list that uses dot bullets.

Unordered list:

  • Item
  • Item
  • Item
  • Item

Ordered list:

  1. Item
  2. Item
  3. Item
  4. Item

Regardless of the type of list, each list contains list elements or <li>.

The third type of list is a description list which uses different tags for its list elements.

The following code snippet respresents how list tags can be utilized:

<ul>
  <li> Unordered list element </li>
  <li> Unordered list element </li>
</ul>
<ol>
  <li> First list element </li>
  <li> Second list element </li>
</ol>

Comment Tag

The comment tag is used to comment parts of the HTML code. The end user will not be able to see the comments unless they have access to the source code.

The follow code snippet represents how comment tags can be utilized:

<!-- This is a comment. Explain to the end user that comments cannot be seen. -->
<p> Comments cannot be seen on the webpage. <p>
